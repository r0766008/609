Wat is een "Wearable"?

Wearables zijn kleding of accessoires met een apparaat dat met een computer verbonden kan worden. Deze toestellen worden hoofdzakelijk gebruikt om het dagelijks leven gemakkelijker te maken. Een wearable kan in verschillende vormen voorkomen zo heb je de bekende "Smartwatches" maar ook in minder bekende vormen. Zo heb je ook de "Smartbril". De "Smartbril geeft je de gevraagde informatie recht voor je hoofd. Dit fenomeen kan je ook vinden in moderne auto's. Een "Head up display" is een soort scherm dat zich voor je bevindt tijdens het rijden en dit geeft je informatie. Wearables worden tegenwoordig het meeste gebruikt voor gezondheid reden. Via een Smartwatch krijg je dan informatie over je gezondheid zo kan je je hartslag zien of bijhouden hoeveel glazen water je hebt gedronken op één dag. Maar ook worden wearables meer als fashion attribuut gebruikt.

Wanneer kan je iets een Wearable noemen?

Je kan iets een wearble noemen als het iets is dat je kan dragen en een apparaat is. Een wearble komt voor in meerdere vormen (zie voorbeelden). Een wearable word vooral gebruikt om je dagelijkse leven gemakkelijker te maken zoals bijvoorbeeld op medisch vlak. Maar het kan ook gebuikt worden om dingen gemakkelijker te maken zo heb je bijvoorbeeld de zelf bindende schoenen. Deze zijn echter nog niet echt op de markt te vinden er zijn wel enkele modelen uitgebracht maar deze waren niet in grootte aantallen gemaakt.
